# aws-serverless-notes-api
Serverless Notes API built with AWS Lambda, API Gateway, and DynamoDB for creating and retrieving notes.
# AWS Serverless Notes API

## Architecture
API Gateway → Lambda → DynamoDB

## Features
- Add notes (POST)
- Get notes (GET)

## Tech Stack
- AWS Lambda
- API Gateway
- DynamoDB
- CloudWatch

## API Endpoint
https://my20vm2vfh.execute-api.us-east-1.amazonaws.com/notes
